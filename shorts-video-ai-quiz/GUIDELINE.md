# 9:16 세로 영상 Safe Zone 가이드라인

Instagram Reels, YouTube Shorts 제작 시 노치/다이나믹 아일랜드 및 UI 오버레이로 인한 콘텐츠 잘림 방지를 위한 제작 가이드입니다.

---

## 1. 기본 영상 규격

| 항목 | 권장 사양 |
|------|----------|
| **해상도** | 1080 × 1920 px |
| **화면비** | 9:16 (세로) |
| **프레임레이트** | 30fps 또는 60fps |
| **포맷** | MP4 (H.264/AAC) |

---

## 2. Safe Zone 개념

Safe Zone은 영상 내에서 플랫폼 UI(좋아요, 댓글, 공유 버튼, 캡션, 프로필 정보 등)와 스마트폰 노치/다이나믹 아일랜드에 가리지 않고 항상 표시되는 영역입니다.

### 잘리는 원인
- 상단: 사용자명, 오디오 정보, 노치/다이나믹 아일랜드
- 하단: 캡션, CTA 버튼, 좋아요/댓글/공유 버튼
- 우측: 인터랙션 아이콘 (좋아요, 댓글, 공유, 저장)
- 좌측: 일부 플랫폼에서 약간의 크롭 발생

---

## 3. 플랫폼별 Safe Zone 규격

### Instagram Reels

```
┌────────────────────────────┐
│     상단 여백: 220px       │  ← 프로필, 오디오 정보
├────────────────────────────┤
│                            │
│         SAFE ZONE          │
│      (1010 × 1380 px)      │
│                            │
│    텍스트/로고/중요 요소    │
│      이 영역 안에 배치      │
│                            │
├──────────────────┬─────────┤
│                  │ 우측    │
│  하단 여백       │ 여백    │
│  320~420px       │ 120px   │
└──────────────────┴─────────┘
```

| 위치 | 여백 (픽셀) | 비고 |
|------|------------|------|
| **상단** | 220px | 프로필, 팔로우 버튼 |
| **하단** | 320px (유기농) / 420px (광고) | 캡션, CTA 버튼 |
| **좌측** | 35~60px | 크롭 방지 |
| **우측** | 120px | 좋아요, 댓글, 공유 아이콘 |

**Safe Zone 영역**: 약 **1010 × 1280~1380 px** (중앙 기준)

### 범용 Safe Zone (모든 플랫폼 호환)

두 플랫폼 모두에서 안전하게 표시되려면 가장 보수적인 기준을 적용합니다:

```
┌────────────────────────────┐
│       상단 여백: 250px     │
├────────────────────────────┤
│                            │
│    ■ 범용 SAFE ZONE ■     │
│      (840 × 1150 px)       │
│                            │
│   모든 중요 요소는         │
│   반드시 이 영역 안에!     │
│                            │
├──────────────────┬─────────┤
│  하단 여백       │ 우측    │
│  420px           │ 190px   │
└──────────────────┴─────────┘
```

| 위치 | 권장 여백 |
|------|----------|
| **상단** | 250px |
| **하단** | 420px |
| **좌측** | 60px |
| **우측** | 190px |

**범용 Safe Zone**: **840 × 1150~1250 px** (중앙 기준)

## 4. 스마트폰 노치/다이나믹 아일랜드 대응

### iPhone (노치 있는 모델)
- 노치 높이: 약 44~47pt (상단 중앙)
- 세이프 에어리어: 상단 47px, 하단 34px 추가 고려

### iPhone (다이나믹 아일랜드 모델 - iPhone 14 Pro 이후)
- 다이나믹 아일랜드: 상단 중앙에 약 126 × 37.3px 크기의 pill 형태
- 영상 시청 시 일반적으로 영향 없지만, 상단 중앙에 중요 요소 배치 피할 것

### Android (펀치홀/노치)
- 기기마다 다양한 크기와 위치
- 일반적으로 상단 20~50px 영역에 위치

**결론**: 상단 250px 여백을 확보하면 대부분의 기기에서 안전

## 5. 제작 체크리스트

### 촬영 단계
- [ ] 카메라/앱 설정을 9:16 세로 모드로 설정
- [ ] 주요 피사체를 화면 중앙에 배치
- [ ] 중요한 동작이 화면 가장자리에서 발생하지 않도록 구성

### 편집 단계
- [ ] 프로젝트 설정: 1080 × 1920px, 30fps 이상
- [ ] Safe Zone 오버레이 템플릿 적용하여 확인
- [ ] 텍스트/자막: 화면 중앙 70% 영역 내 배치
- [ ] 로고/워터마크: Safe Zone 내부에 배치
- [ ] CTA(행동 유도): 하단 UI와 겹치지 않도록 위쪽에 배치

### 텍스트 배치 권장사항
| 요소 | 권장 위치 |
|------|----------|
| **제목/헤드라인** | 상단에서 300px 아래 시작 |
| **본문 자막** | 화면 중앙 또는 하단에서 500px 위 |
| **CTA 텍스트** | 하단에서 최소 450px 위 |
| **로고** | Safe Zone 내 모서리 (상단 좌측 권장) |

### 내보내기 단계
- [ ] 해상도: 1080 × 1920px
- [ ] 포맷: MP4 (H.264)
- [ ] 비트레이트: 적정 수준 유지 (10-20Mbps 권장)
- [ ] 여러 기기에서 미리보기 테스트

## 6. 편집 도구별 Safe Zone 적용 방법

### Premiere Pro / DaVinci Resolve
1. PNG 오버레이 템플릿 다운로드
2. 미디어 풀에 추가
3. 타임라인 상단 레이어에 배치
4. 편집 시 가이드로 활용
5. 최종 내보내기 전 삭제

### CapCut / 모바일 편집앱
1. 오버레이 이미지 가져오기
2. 스티커/레이어로 추가
3. 투명도 조절하여 가이드로 사용
4. 완성 후 레이어 삭제

### Canva
1. 9:16 세로 템플릿 선택
2. 중앙 영역에 요소 배치
3. 가장자리 마진 유지

## 7. 흔한 실수 및 해결방법

| 실수 | 문제점 | 해결방법 |
|------|--------|----------|
| 텍스트를 화면 하단에 배치 | 캡션/버튼에 가려짐 | 하단에서 450px 이상 위로 이동 |
| 로고를 우측 하단에 배치 | 공유/저장 버튼에 가려짐 | 좌측 상단 또는 Safe Zone 내부로 이동 |
| 중요 정보를 상단 가장자리에 배치 | 노치/상태바에 가려짐 | 상단에서 250px 아래로 이동 |
| 자막이 화면 가장자리까지 확장 | 좌우 크롭 시 잘림 | 좌우 100px 이상 여백 확보 |
| 16:9 영상을 그대로 업로드 | 상하 검은 바 발생, 몰입감 저하 | 9:16으로 리프레이밍 |

## 8. 피드/그리드 미리보기 대응

### Instagram 피드 표시
- 피드에서는 **4:5 비율**로 크롭되어 표시
- 중앙 1080 × 1350px 영역이 피드 미리보기로 노출
- 썸네일/커버에 중요한 요소는 4:5 영역 안에 배치

### YouTube Shorts 그리드
- 채널 페이지에서 **1:1 정사각형**으로 표시될 수 있음
- 썸네일 중앙 1080 × 1080px 영역 고려

### 권장사항
- 핵심 비주얼/텍스트는 **화면 정중앙 1080 × 1080px** 영역에 배치
- 이 영역이 모든 미리보기에서 안전하게 표시됨
